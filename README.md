# NoteWithMe

Aplicacion web responsive que permite crear notas, editarlas y eliminarlas. 


funcionalidades básicas:

* Autenticación con correo y/o Google y/o Facebook.
* Cerrar sesión.
* Crear una nueva nota.
* Ver mis notas.
* Editar mis notas.
* Borrar mis notas.

## Historias de usurio 
1. Usuaria/o debe poder crear una cuenta o iniciar sesión 
2. Usuaria/o debe poder tomar nota
3. Usuaria/o debe poder editar o eliminar sus notas
4. Usuaria/o debe poder encontrar sus notas


## Criterios de aceptación
Lo que debe ocurrir para que se satisfagan las necesidades del usuario

* Anotar un título a mi nota
* Agregar el contenido de mi nota.
* Ver todas mis notas.
* Ver la última modificación de la nota.
* Modificar las notas.
* Eliminar notas.
* Se ve y funciona bien en una _Tablet_
